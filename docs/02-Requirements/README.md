# Software Requirements & Analysis

This folder contains requirements documentation and user stories for the system.

## Contents

- **EduTrack – Software Requirements Specification (SRS).pdf** - Comprehensive SRS document covering functional and non-functional requirements
- **EduTrack – User Stories.pdf** - User stories, use cases, and user workflows for all system features

## Purpose

These documents provide business analysts, product managers, and developers with clear requirements and user expectations. They define what the system should do and how users will interact with it.

## Audience

- Business Analysts
- Product Managers
- Developers
- Quality Assurance Engineers
- Project Managers
