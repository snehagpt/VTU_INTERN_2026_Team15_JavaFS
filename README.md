# 📚 EduTrack – Student Performance & Academic Tracking System

**EduTrack** is a modern **student performance and academic tracking platform** designed to help educational institutions efficiently monitor student progress, manage academic records, and improve communication between students, teachers, and administrators.

The system enables users to track **attendance, marks, assignments, and overall academic performance** through a **centralized digital platform**, making academic management simpler and more transparent.

---

# 🎯 Project Objective

The objective of **EduTrack** is to simplify academic management by providing a digital platform that helps institutions:

✔ Track student performance
✔ Maintain academic records efficiently
✔ Improve transparency between students, teachers, and administrators
✔ Reduce manual paperwork and administrative workload

---

# ⚙️ Tech Stack

💻 **Frontend:** React.js

🗄 **Database:** MySQL
🧪 **API Testing:** Postman
🔧 **Version Control:** Git & GitHub

# ⭐ Key Features

# 🗂 Database Entities

The EduTrack database includes the following core tables:

👨‍🎓 **Students**
👩‍🏫 **Teachers**
📘 **Courses**
📅 **Attendance**
📝 **Assignments**
📊 **Marks**
📢 **Announcements**

All tables are connected using **Primary Keys and Foreign Keys** to ensure **data consistency, integrity, and efficient data management**.

---

# 🔐 Authentication Module

The authentication system includes:

✔ User Signup API
✔ User Login API
✔ Input Validation
✔ Secure Authentication Handling
✔ User Role Verification
✔ Database Connectivity Validation

All APIs are **tested using Postman** to ensure reliability and correct functionality.
