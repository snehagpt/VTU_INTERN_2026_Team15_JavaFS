# EduTrack – Performance & Attendance Analytics

EduTrack is a comprehensive performance and attendance analytics platform built for educational institutions. It brings together attendance tracking, marks management, and performance analytics into a single, real-time platform designed for admins, teachers, and students.

---

## 🚀 Project Objective

The goal of EduTrack is to simplify the process of tracking student progress by providing a centralized digital platform for managing attendance, academic marks, and real-time performance insights.

---

## 🛠️ Tech Stack

**Frontend:** React  
**Backend:** Spring Boot  
**Database:** MySQL  
**API Testing:** Postman  
**Version Control:** Git & GitHub  

---

## 📌 Key Features

* Secure User Authentication (Login/Signup)
* Role-Based Access (Admin / Teacher / Student)
* Real-Time Attendance Tracking
* Marks and Grade Management
* Interactive Performance Analytics and Trend Reports
* OTP-Based Password Reset via Email
* CSV Bulk Upload for Students and Academic Records

---

## 📂 Database Entities

The system database includes the following core tables:

* Users
* Students
* Courses
* Attendance
* Marks
* OTP

These tables are connected using primary and foreign key relationships to maintain data integrity.

---

## 🔐 Authentication Module

The authentication system provides:

* User Signup API
* User Login API
* OTP Verification for Secure Access
* Session Management with JWT Tokens
* Role-Based Dashboard Redirection

All APIs are tested and verified using Postman.
